# Experiments

This directory is used to host the output files, checkpoints and logs generated during the experiments (both training and evaluation).

Directory structure:
```
|- experiments/
  |- {experiment_series_id}/
    |- results.csv
    |- {model_id}
      |- {configuration_id}
      |- ...
    |- ...
  |- ...
```
